ls -> lista arquivos
ls -a -> lista arquivos ocultos
cd [diretorio] -> move entre diretorios
pwd -> mostra diretorios de trabalho
mkdir [nome pasta] -> cria uma pasta no diretório 
rm -rf [nome pasta] -> deleta uma pasta no diretório 
cat [nome arquivo] -> mostra conteúdo do arquivo
git init . -> transforma a pasta em repositório de git
ctrl + l -> limpa terminal
