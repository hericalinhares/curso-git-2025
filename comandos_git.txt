git init . -> cria um repo git na pasta atual
git status -> mostra o status do trabalho (de cada arquivo)
git add nome_arquivo -> adiciona arquivo a ser commitado
git commit -m "msg" -> commita os arquivos adicionados
git log -> mostra todo o histÃ³rico de commit
git remote -v -> verifica o remote atual
git remote remove origin -> remove remote atual
git push origin branchname -> envia para o github
git pull origin branchname -> carrega o arquivo do github
git fetch origin branchname -> carrega todos os branchs
git clone gitaddress -> cria um repositorio a partir do github
